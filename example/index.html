aMail jquary plugin of email protection.

Name(at)domain(dot)com

JavaScript

	<script type="text/javascript" src="jquery-1.3.2.min.js"></script>
	<script type="text/javascript" src="jquery.amail.js"></script>
	<script type="text/javascript">
		$(document).ready(function(){$('.yourclass').amail('(at)','(dot)');});
	</script>

html

	<a class="yourclass" title="my email">Name(at)domain(dot)com</a>

result html

	<a class="yourclass" href="mailto:Name@domain.com" title="my email">my email</a>
The plugin replace "(at)" to "@" and replace "(dot)" to "." . You can make you own protection. you can change "(at)" and "(dot)".
